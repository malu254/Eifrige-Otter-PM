Technische Anforderungen:

- Nutzer müssen sich einloggen können (Security)

- Arbeitszeiten müssen exportiert werden können

- Es muss eine API geben, um auf das System zuzugreifen

- Software sollte in HTML/CSS/JavaScript geschrieben werden

- Es benötigt einen lauffähigen Web-Server um die Zeiterfassung zu hosten

- Plattformunabhängige Software (Windows, macOS)

- Verwendung einer Lizenzfreien Datenbank

- Die Zeiterfassung muss performant laufen

- Sichere Datenübertragung via HTTPS

- Es muss eine Versionsverwaltung geben (GIT)

- Die Weboberfläche muss skalierbar sein

- Es muss erkannt werden, ob ein Nutzer schon arbeitet und Doppeleingaben blockieren

- Mitarbeitern muss ihr Arbeitszeitmodell zugewiesen werden können.



Funktionale Anforderungen:

- Die Nutzer müssen benachrichtigt werden, wenn sie ihre Arbeitszeit überschreiten

- Nutzer müssen angelegt werden können (HR)

- Nutzer müssen Passwort ändern können. (Mitarbeiter, HR)

- Admin kann Nutzer bearbeiten, löschen, hinzufügen

- Sprache muss gewechselt werden können, ohne Neustart

- Es muss Terminals in den Büroräumen geben

- Nutzer erhalten Erinnerung, wenn sie ihre Arbeitszeit überschreiten/unterschreiten

- Das System muss erkennen, ob die Ruhezeit nach Arbeitsschutzgesetz eingehalten wurde (mind. 11h)

- Die maximale Wochenarbeitszeit sollte nur in einem bestimmten Rahmen ohne Genehmigung überschritten werden können

- Das System muss Wochenende und Feiertage blockieren zum stempeln

- Man soll sich nur zwischen 6 und 22 Uhr stempeln können, um betriebliche Arbeitszeit zu sammeln

- Stempelt man außerhalb dieser Zeit muss das anders dokumentiert werden.

- Das System muss den Nutzer darauf hinweisen, 30min Pause zu machen wenn er mehr als 6 Stunden gearbeitet hat.

- Das System muss den Nutzer darauf hinweisen, dass er zusätzliche 15min Pause machen muss, wenn er mehr als neun Stunden arbeitet.

- Die Pause muss von der Arbeitszeit automatisch abgezogen werden, sodass die Mitarbeiter nicht händisch stempeln müssen

- Mitarbeiter unter 13 dürfen nicht angelegt werden

- Mitarbeiter zwischen 13-15 dürfen maximal 2 Stunden am Tag arbeiten

- Mitarbeiter von 15-18 dürfen maximal 8 Stunden am Tag arbeiten

- Mitarbeiter von 15-18 dürfen nur zwischen 6 und 20 Uhr stempeln.    12/34/21/34/11

- Mitarbeiter zwischen 15-18 müssen nach 4,5 Stunden 30min Pause machen.

- Mitarbeiter zwischen 15-18 müssen nach 6h mindestens 60min Pause machen.

- Mitarbeiter zwischen 15-18 dürfen nur 1h nach Arbeitsbeginn und 1 Stunde vor Arbeitsende Pause nehmen.


Constraints:

- Das Projekt muss innerhalb von 24 Wochen erledigt sein

- Das Arbeitszeitschutzgesetz muss eingehalten werden

- Das Jugendarbeitsgesetz muss eingehalten werden

- Meilensteine und Abgabetermine sind einzuhalten

- Alles muss vollständig dokumentiert sein

- Arbeitszeiten außerhalb der Kernarbeitszeit von 6-22 Uhr müssen auch dokumentiert werden

- Benachrichtigungen müssen klar erkennbar sein und dürfen nicht deaktiviert werden können.
